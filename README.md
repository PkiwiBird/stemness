### Comparison and functional/network analysis of stemness gene sets
This repo contains the R scripts of the analysis done in "Barata, T.; Duarte, I.; Futschik, M.E. Integration of Stemness Gene Signatures Reveals Core Functional Modules of Stem Cells and Potential Novel Stemness Genes. Genes 2023, 14, 745. https://doi.org/10.3390/genes14030745"
